# Navigator

## For creating application are used the following technologies

```bash
Kotlin programming language - developed by JetBrains, native language for android Application since 2017
Kotlin Coroutines - developed by Kotlin community for making async programming lightweight and easy
Retrofit - for fetching remote data
Dagger 2 - created by Google DI(Dependency injection) library, using annotationProcessing
MVVM design pattern - for reactive UI, for separated UI and business logic
S.O.L.I.D principles - for implementing reusable and testable code
Android Jetpack Architectural component - created by Google for implementing Reactive programming

UI automator - for UI testing
```
